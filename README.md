# odin-etch-a-sketch
**Summary:**
- A browser drawing tool that lets users draw on a canvas constructed from many divs.
- Features:
  - Clear
  - Draw
  - Rainbow
  - Erase
  - Colour selection
  - Adjust grid size

**Tech Used:**
- HTML
- CSS
- JS
